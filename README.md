<h1 align="center">untitled-whiteboard</h1>

An untitled whiteboard app runs on Web.

## 运行

如果没有安装 `pnpm`，请先安装 `pnpm` 

```shell
npm i -g pnpm
```

安装依赖

```shell
pnpm i
```

运行 development 环境

```shell
pnpm dev
```
